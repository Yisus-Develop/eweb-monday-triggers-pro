# EWEB Monday Triggers Pro

Advanced Webhook Trigger engine for **Monday.com**. This plugin enables real-time synchronization between WordPress events (Form submissions, status changes, user actions) and Monday.com boards. Part of the **EWEB Plugin Suite**.

## 🚀 Key Features

- **Real-time Synchronization:** Instantly trigger Monday.com webhooks based on WordPress actions.
- **Custom Event Hooks:** Easily map any WordPress hook to a specific Monday.com board action.
- **Payload Management:** Cleanly format and send data packets to Monday.com.
- **Async Execution:** Optimized to run in the background without affecting user experience.
- **GitHub Integration:** Automatic updates directly from the repository.

## 🛠️ Installation

1. Upload the `monday-webhook-trigger` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your webhook URLs in the **Monday Triggers** settings.
